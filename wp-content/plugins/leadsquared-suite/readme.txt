== Description ==
LeadSquared offers a suite of plugins that help in getting best possible integration of your WordPress site with LeadSquared app. 

The suite is collection of following plugins:

= 1)	LeadSquared Website Tracking for WordPress: = This plugin will install a script on pages of your website pages. This script will track visitor’s online behavior like page visits, time spent, link clicks. It will also keep track of source parameters of the visitor. 
= 2)	LeadSquared Forms in WordPress: =  This module allows you to use forms created in LeadSquared in your WordPress account to capture leads.
= 3)	Comments to LeadSquared: = It allows capturing leads form WordPress post’s comments. For existing leads, it helps you to track each comment as an activity under Lead Activity History. If the comment is with a new lead, the new lead is added in LeadSquared and the comment is tracked under Activity history of that new lead.
= 4)	Contact Form 7 to LeadSquared: =  Using this plugin you can capture leads from your Contact Form 7 to LeadSquared. 


== Installation ==

In your WordPress Dashboard, from left panel select **Plugins->Add New**
then SELECT the zip file of **leadsquared-suite.zip** hit INSTALL
After a few seconds there is an option for **Activate Plugin**
click the link of Activate Plugin. 

== Frequently Asked Questions ==

= Can I Install the plugin manually ? =

Yes, you can. Extract the zip file of the "leadsquared-form.zip" then place the whole folder in the following location
"yousite\wp-content\plugins\".

= Do I need WordPress newer than 4.0 ? =

Ans - Yes, your WordPress version should be greater than 4.0

= How do I uninstall? =

Ans - Go to the Uninstall tab and then click the Uninstall button at the bottom of the page. You'll be sent to the WordPress Plugins page. Now deactivate and uninstall the Plugin
